# 🐕 八重之犬 · HIT-HOG

> 一款融合 Roguelite 元素的浏览器动作游戏，灵感源自《哈迪斯》美学风格。扮演一只身怀绝技的狗，踢飞、劈砍、闪避，在无尽的敌浪中杀出一条血路。

## ✨ 特性

- **双模式玩法** — 经典剧情波次模式 & 无尽割草模式，风格截然不同
- **丰富技能系统** — 踢击、劈砍、撒娇、嘲讽四大技能，各具奇效
- **Roguelite 升级** — 每局随机升级选项，构建专属流派（爆发、吸血、速攻、AOE……）
- **武器进化** — 无尽模式中武器可升级进化，解锁强力形态
- **遗物系统** — 收集遗物叠加 Buff，雪球越滚越大
- **BOSS 战** — 多阶段 Boss 机制，血量阈值触发特殊行为
- **成就系统** — 解锁多项成就，记录高光时刻
- **排行榜** — 在线排名，与其他玩家争夺榜首
- **商店 & 永久强化** — 骨头货币购买永久升级，跨局成长
- **PC + 移动端支持** — 键盘操作与触屏摇杆双适配

## 🕹️ 操作方式

### 键盘（PC）

| 按键 | 操作 |
|------|------|
| `W` / `↑` | 跳跃 |
| `A` / `←` | 向左移动 |
| `D` / `→` | 向右移动 |
| `J` | 踢击 |
| `K` | 劈砍 |
| `L` | 撒娇（对 Boss 额外伤害） |
| `U` | 嘲讽（吸引仇恨） |
| `空格` | 格挡 |
| `ESC` | 暂停 |

### 触屏（移动端）

屏幕左侧虚拟摇杆控制移动，右侧技能按钮点击使用技能。

## 🎯 游戏模式

### 经典模式
共 10 波敌人，逐波推进。游戏途中触发天赋选择与随机升级事件，挑战最终 Boss。

### 无尽模式（割草模式）
敌人持续刷新，强度随时间阶段递增。通过击杀积累经验值，升级解锁新武器与技能，体验 Vampire Survivors 式的割草快感。

## ⚔️ 技能说明

| 技能 | 说明 |
|------|------|
| 🦵 踢击 | 基础攻击，可穿透多个敌人，升级后伤害与范围大幅提升 |
| ⚔️ 劈砍 | 强力单次攻击，冷却较长，可升级为双劈 |
| 💅 撒娇 | 对 Boss 造成大量额外伤害，可附加无敌帧 |
| 📣 嘲讽 | 吸引范围内所有敌人仇恨，配合群体技能使用效果绝佳 |

## 🚀 快速开始

本游戏为纯前端单文件项目，无需安装任何依赖。

```bash
# 克隆仓库
git clone https://github.com/Hygubline/game-HIT-HOG.git

# 直接用浏览器打开
open index.html
```

或将 `index.html` 部署至任意静态托管服务（GitHub Pages、Vercel 等）即可在线游玩。

---

## 🛠️ 技术栈

- **语言**：HTML5 + CSS3 + Vanilla JavaScript
- **渲染**：Canvas 2D API
- **数据存储**：localStorage（本地存档）+ Supabase（在线排行榜）
- **无任何外部框架依赖**，单文件即完整游戏
