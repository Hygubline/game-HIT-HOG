# 八重之犬 - 开发者日志

## 2026-03-20

### v0.3.1 - 技能栏UI
- 添加 PC 端技能栏显示（屏幕底部居中）
- 显示 5 个技能：踢击(J)、剑气(K)、冲刺(O)、撒娇(U)、嘲讽(I)
- 实时显示技能冷却时间（秒）
- 技能就绪时边框变绿，冷却中变灰
- 手机端继续使用触屏按钮

### v0.3.0 - 素材更新
- 添加 4 个小怪图片 (`moster1.png` ~ `moster4.png`)，替代原有圆形绘制
- 添加 2 个 Boss 图片 (`boss1.png`, `boss2.png`)，随机选择显示
- 敌人根据移动方向自动翻转朝向
- 图片缩放比例：
  - 普通敌人: 0.08 ~ 0.1
  - 特殊敌人: 0.07 ~ 0.12
  - Boss: 0.18

### v0.2.0 - UI/菜单优化
- 添加游戏内暂停功能
  - 右上角暂停按钮
  - 按 ESC 键暂停/继续
  - 退出确认弹窗
- 优化开始菜单流程
  - 点击"开始游戏"后隐藏主菜单，显示模式选择
  - 添加"返回"按钮回到主菜单
- 添加 3 张地图背景图片 (`map1.png` ~ `map3.png`)
  - 峡谷夜战 → map1.png
  - 广场舞战场 → map2.png
  - 宿命天台 → map3.png

### v0.1.1 - Bug修复
- 修复角色朝向问题：往左走时角色现在正确朝左

### v0.1.0 - 初始版本
- 基础游戏框架
- 三种游戏模式：速战模式、无尽模式、波次挑战
- 角色技能：踢击、剑气、撒娇、冲刺
- 多种敌人类型：普通怪、冲锋怪、嘴炮怪、坦克怪、碰瓷怪
- Boss 战斗系统
- 升级系统和天赋选择
- 商店和成就系统
- 支持键盘和触屏操作
