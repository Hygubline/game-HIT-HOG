# 八重之犬 - 开发者日志

## 2026-03-20

### v0.4.0 - 核心玩法重构 (KICK-FOCUSED DESIGN)
**设计目标**: 以"踢飞敌人产生连锁反应"为核心乐趣

**简化系统:**
- 移除嘲讽(Taunt)技能，效果合并到撒娇
- 隐藏波次模式，保留速战和无尽两种模式
- 坦克敌人生成率降低，更强调行动压力
- 遗物只从Boss掉落（不再从精英掉落）
- 随机事件替换为"危机时刻"系统

**强化踢击:**
- 新增"处决踢"：弱化/低血敌人被踢时获得2.5倍击退
- 处决踢触发短暂慢动作和金色粒子特效
- 踢飞的敌人标记为"投射物"，连锁伤害更高

**连锁碰撞升级:**
- 连锁反应现在给予额外分数奖励
- 连锁文字升级：连锁! → 双杀! → 三连! → 超神! → 绝杀!
- 3连锁以上显示大事件提示并给予500分奖励
- 连锁时屏幕震动随连击数增加

**撒娇重做 (单一控场技能):**
- 合并嘲讽效果：敌人被定身后会冲向玩家
- 不再秒杀小怪，而是将其"弱化"到1血
- 弱化的敌人完美适合处决踢 → 制造连锁！
- Boss也会被短暂定身

**危机时刻系统:**
- 替代混乱的随机事件
- 三种危机：冲锋潮、嘴炮狂潮、爆炸预警
- 存活后获得500分奖励和30%几率回血

**Boss改进:**
- Boss召唤的小弟现在是"弱化"状态
- 更适合踢向Boss造成伤害
- 更大的Boss击败奖励(+2000分)

### v0.3.4 - 主菜单布局优化
- 改为左右双栏布局：左边文字按钮，右边角色图
- 减弱动画效果：标题glow更柔和，角色浮动幅度减小
- 统一次要按钮风格：商店/成就/教程使用相同样式
- 调整统计栏紧凑度

### v0.3.3 - 主菜单UI重设计
- 全新专业级主菜单设计
- 添加毛玻璃效果的菜单面板
- 新增统计栏：最高分、游戏局数、总击杀
- 更清晰的视觉层次：主按钮突出，次要按钮低调
- 模式选择改为卡片式设计
- 更优雅的动画效果（柔和浮动代替跳动）
- 标题发光效果

### v0.3.2 - 音效系统
- 使用 Web Audio API 生成电子音效（无需音效文件）
- 添加 12 种音效：
  - 踢击、剑气、撒娇、冲刺、嘲讽
  - 跳跃、受伤、击杀、连击
  - Boss出场、升级、游戏结束、拾取道具
- 连击音效会随连击数升高音调
- 添加音效开关按钮（右上角 🔊/🔇）

### v0.3.1 - 技能栏UI
- 添加 PC 端技能栏显示（屏幕底部居中）
- 显示 5 个技能：踢击(J)、剑气(K)、冲刺(O)、撒娇(U)、嘲讽(I)
- 实时显示技能冷却时间（秒）
- 技能就绪时边框变绿，冷却中变灰
- 手机端继续使用触屏按钮
