# 八重之犬 - 开发者日志

## 2026-03-20

### v0.4.6 - 格挡技能与Boss连锁Bug修复
**新增格挡技能:**
- 按P键(PC端)或点击🛡️按钮(手机端)发动格挡
- 格挡时完全免疫所有伤害
- 格挡期间无法移动或使用其他技能
- 格挡持续约0.75秒
- 成功格挡攻击后获得短暂无敌并进入3秒冷却
- 格挡时显示绿色护盾特效

**Bug修复 - Boss连锁秒杀:**
- 修复踢飞的敌人撞到Boss时直接秒杀Boss的严重Bug
- 原因：连锁碰撞代码直接将被撞敌人HP设为0，未检查是否为Boss
- 修复后：Boss受到连锁碰撞时只扣除伤害(5+连锁数×2)，不会被秒杀
- Boss血条正确更新，死亡时正常触发击杀流程

### v0.4.2 - 在线排行榜系统 (SUPABASE LEADERBOARD)
**设计目标**: 让玩家可以与朋友比较分数

**Supabase 集成:**
- 使用 Supabase 作为后端数据库
- 客户端直连，适合静态网站 (GitHub Pages)
- 异步加载，不影响游戏性能

**排行榜功能:**
- 主菜单新增"排行榜"按钮
- 支持速战模式和无尽模式双榜单
- 显示 Top 10 玩家排名
- 展示：排名、玩家名、分数、击杀、最大连击、存活时间

**分数提交:**
- 游戏结束后可输入昵称
- 点击"提交分数"上传到云端
- 昵称自动保存，下次游戏自动填充
- 提交状态反馈（成功/失败提示）

**数据结构:**
- name: 玩家昵称 (最多12字符)
- mode: 游戏模式 (speed/endless)
- score: 分数
- kills: 击杀数
- max_combo: 最大连击
- survival_time: 存活时间

**UI设计:**
- 排行榜模态框，深色半透明背景
- 标签页切换速战/无尽模式
- 金/银/铜色高亮前三名
- 游戏结束界面内嵌提交区域

### v0.4.1 - 打击感与游戏手感优化 (JUICE PASS)
**设计目标**: 让攻击更有冲击力，让踢击成为招牌动作

**帧冻结系统 (Hitstop):**
- 新增轻量级帧冻结：攻击命中时游戏短暂暂停
- 分级冻结：轻击(2帧) → 重击(4帧) → 处决(6帧) → 连锁(8帧)
- 让每次命中都有"击中感"

**攻击反馈分层:**
- 普通攻击：轻微震屏(3)、小型粒子
- 重击：中等震屏(10)、更多粒子、敌人后退
- 处决踢：强震(18)、金色爆发粒子、冲击波环、慢动作

**敌人受击反应:**
- 新增白色闪烁效果(8帧)
- 新增短暂硬直(4-12帧)
- 新增后退反冲(根据攻击类型)
- Boss对轻击免疫反应，重击/处决仍有反馈

**连锁碰撞增强:**
- 连锁命中触发帧冻结
- 3连锁以上触发慢动作
- 更大的动量传递(1.3x)
- 新增"chain"连锁专属音效

**音效增强:**
- 新增"impact"重击音效 - 低沉有力
- 新增"execution"处决音效 - 双层叠加
- 新增"chain"连锁音效 - 清脆上升

**视觉特效优化:**
- 强力命中产生白色闪光爆发
- 处决踢产生金色冲击波环
- 粒子数量和大小随攻击力缩放
- 敌人被击中时白色高光叠加

### v0.4.0 - 核心玩法重构 (KICK-FOCUSED DESIGN)
**设计目标**: 以"踢飞敌人产生连锁反应"为核心乐趣

**简化系统:**
- 移除嘲讽(Taunt)技能，效果合并到撒娇
- 隐藏波次模式，保留速战和无尽两种模式
- 坦克敌人生成率降低，更强调行动压力
- 遗物只从Boss掉落（不再从精英掉落）
- 随机事件替换为"危机时刻"系统

**强化踢击:**
- 新增"处决踢"：弱化/低血敌人被踢时获得2.5倍击退
- 处决踢触发短暂慢动作和金色粒子特效
- 踢飞的敌人标记为"投射物"，连锁伤害更高

**连锁碰撞升级:**
- 连锁反应现在给予额外分数奖励
- 连锁文字升级：连锁! → 双杀! → 三连! → 超神! → 绝杀!
- 3连锁以上显示大事件提示并给予500分奖励
- 连锁时屏幕震动随连击数增加

**撒娇重做 (单一控场技能):**
- 合并嘲讽效果：敌人被定身后会冲向玩家
- 不再秒杀小怪，而是将其"弱化"到1血
- 弱化的敌人完美适合处决踢 → 制造连锁！
- Boss也会被短暂定身

**危机时刻系统:**
- 替代混乱的随机事件
- 三种危机：冲锋潮、嘴炮狂潮、爆炸预警
- 存活后获得500分奖励和30%几率回血

**Boss改进:**
- Boss召唤的小弟现在是"弱化"状态
- 更适合踢向Boss造成伤害
- 更大的Boss击败奖励(+2000分)
